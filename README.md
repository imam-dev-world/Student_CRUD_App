# Student_CRUD_App

A simple REST API built with Spring Boot to manage student records.

## Tech Stack
- Java 17
- Spring Boot 4.0.6
- Spring Data JPA
- MySQL
- Maven

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | /create | Create student |
| PUT | /update | Update student |
| GET | /getStudent/{id} | Get student by ID |
| DELETE | /deletestudent/{id} | Delete student |
